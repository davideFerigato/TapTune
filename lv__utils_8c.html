Searches base[0] to base[n - 1] for an item that matches *key.

Note
The function cmp must return negative if its first argument (the search key) is less than its second (a table entry), zero if equal, and positive if greater.
Items in the array must be in ascending order.
Parameters
keyPointer to item being searched for
basePointer to first element to search
nNumber of elements
sizeSize of each element
cmpPointer to comparison function (see unicode_list_compare as a comparison function example)
Returns
a pointer to a matching item, or NULL if none exists.
